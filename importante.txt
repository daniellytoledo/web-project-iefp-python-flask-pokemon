ACESSAR O CMD DO WINDOWS E ENTRAR NA PASTA DO PROJETO:
cd \wamp64\www\web-project-iefp-python-flask-pokemon

PASSO A PASSO (
    1. criar env pelo cmd        -> python -m venv venv (só uma vez)
    2. ativar o venv             -> venv\Scripts\activate
    3. instalar o flash pelo cmd -> pip install flask
    4. instalar pymysql pelo cmd -> pip install pymysql
    5. instalar dotenv pelo cmd  -> pip install python-dotenv
    6. criar a estrutura         -> (

        projeto 
        │
        ├── app/
        │   ├── models/
        │   │   ├── rep_nome_db 
        │   ├── routes/
        │   │   ├── rt_nome_db
        │   ├── services/
        │   │   ├── srv_nome_db 
        │   ├── static/
        │   │   ├── css/
        │   │   ├── fonts/
        │   │   ├── imgs/
        │   │   ├── js/
        │   ├── templates/
        │   │
        │   ├── __init__.py
        │   │
        │   ├── bd_config.py
        │
        ├── venv/ # já foi criado no passo 1
        │
        ├── run.py
        │
        ├── .env
        │ 
        ├── .gitignore

    7. Criar rep(models), serviços, rota (

        models   -> [consulta/atualiza o banco (representam e manipulam a manipulação de dados da aplicação (as tabelas de dados), buscam ou salvam dados no banco)] 

        serviços -> [prepara o pedido (implementam a lógica do negócio e processam as regras da aplicação, ou seja, eles decidem o que fazer)]

        rotas    -> [recebe opedido (recebem requisições HTTP, chamam os serviços e retornam a resposta para o cliente)]

        model → serviço → rota → template

    )

    8. Criar templates (
        -> home.html     (página inicial)
        -> base.html     (página que pode ser usada como base para outras)
        -> detalhes.html (página que recebe mais informações do BD)
        -> e outras 
    )

    9. python run.py no cmd para gerar o link para acessar no navegador

    )
)

CRIAR LINK LOCALHOST NO CMD PARA WEB:
python run.py

CRIAR UTILIZADOR ADMIN PRIMEIRO:
python -m app.scripts.criar_utilizador
